Project Python Foundations: FoodHub Data Analysis

Marks: 60

Context

The number of restaurants in New York is increasing day by day. Lots of students and busy professionals rely on those restaurants due to their hectic lifestyles. Online food delivery service is a great option for them. It provides them with good food from their favorite restaurants. A food aggregator company FoodHub offers access to multiple restaurants through a single smartphone app.

The app allows the restaurants to receive a direct online order from a customer. The app assigns a delivery person from the company to pick up the order after it is confirmed by the restaurant. The delivery person then uses the map to reach the restaurant and waits for the food package. Once the food package is handed over to the delivery person, he/she confirms the pick-up in the app and travels to the customer's location to deliver the food. The delivery person confirms the drop-off in the app after delivering the food package to the customer. The customer can rate the order in the app. The food aggregator earns money by collecting a fixed margin of the delivery order from the restaurants.

Objective

The food aggregator company has stored the data of the different orders made by the registered customers in their online portal. They want to analyze the data to get a fair idea about the demand of different restaurants which will help them in enhancing their customer experience. Suppose you are hired as a Data Scientist in this company and the Data Science team has shared some of the key questions that need to be answered. Perform the data analysis to find answers to these questions that will help the company to improve the business.

Data Description

The data contains the different data related to a food order. The detailed data dictionary is given below.

Data Dictionary

  • order_id: Unique ID of the order
  • customer_id: ID of the customer who ordered the food
  • restaurant_name: Name of the restaurant
  • cuisine_type: Cuisine ordered by the customer
  • cost: Cost of the order
  • day_of_the_week: Indicates whether the order is placed on a weekday or weekend (The weekday is from Monday to Friday and the weekend is Saturday and Sunday)
  • rating: Rating given by the customer out of 5
  • food_preparation_time: Time (in minutes) taken by the restaurant to prepare the food. This is calculated by taking the difference between the timestamps of the restaurant's order confirmation and the delivery person's pick-up confirmation.
  • delivery_time: Time (in minutes) taken by the delivery person to deliver the food package. This is calculated by taking the difference between the timestamps of the delivery person's pick-up confirmation and drop-off information

def show_graph(data, graph_name, column_label, color):
    """
    Display a histogram, boxplot, or countplot based on a specific graph type.

    Parameters:
    - data (Dataframe): The dataset containing the column to be plotted.
    - graph_name (str): Type of graph to display ('histogram_boxplot'
                        or 'countplot').
    - column_label (str): The name of the column to plot.
    - color (str): Color for the plot elements.
    """

    # Map column labels to titles
    title_map = {
        'order_id': "Order ID",
        'customer_id': "Customer ID",
        'restaurant_name': "Restaurant Name",
        'cuisine_type': "Cuisine Type",
        'cost_of_the_order': "Cost of the Order",
        'day_of_the_week': "Day of the Week",
        'rating': "Rating",
        'food_preparation_time': "Food Preparation Time",
        'delivery_time': "Delivery Time"
    }

    # Get the title for the specified column, or handle invalid column
    title = title_map.get(column_label, None)
    if title is None:
        print(f"Enter valid column label!")
        return

    # Display both a histogram and boxplot for the specified column
    if graph_name == 'histogram_boxplot':

        # Set up a figure with 1 row and 2 columns for the plots
        fig, (ax1, ax2) = plt.subplots(1, 2, figsize = (12, 5))

        # Plot the histogram
        sns.histplot(
            data = data,
            x = column_label,
            bins = 43,
            color = color,
            ax = ax1
        )
        ax1.set_title(f'Histogram of {title}')
        ax1.set_xlabel(title)
        ax1.set_ylabel('Frequency')

        # Plot the boxplot
        sns.boxplot(
            data = data,
            x = column_label,
            color = color,
            ax = ax2
        )
        ax2.set_title(f'Boxplot of {title}')
        ax2.set_xlabel(title)

        # Adjust layout to prevent overlap and display the plots
        plt.tight_layout()
        plt.show()

        # Calculate and print descriptive statistics for the column
        Q1 = data[column_label].quantile(0.25)  # 25th percentile
        Q2 = data[column_label].median()        # Median (50th percentile)
        Q3 = data[column_label].quantile(0.75)  # 75th percentile
        IQR = Q3 - Q1                           # Interquartile range
        upper_bound = Q3 + 1.5 * IQR            # Upper bound for outliers
        lower_bound = Q1 - 1.5 * IQR            # Lower bound for outliers

        # Print the statistical summary
        print(
            f"Details for {title}:\n"
            f"25th percentile: {Q1:.2f}\n"
            f"Median (50th percentile): {Q2:.2f}\n"
            f"75th percentile: {Q3:.2f}\n"
            f"Upper Bound: {upper_bound:.2f}\n"
            f"Lower Bound: {lower_bound:.2f}\n"
            f"Interquartile Range (IQR): {IQR:.2f}\n"
        )

    # Display a countplot for the specified column
    elif graph_name == 'countplot':
        # Set figure size for countplot
        plt.figure(figsize = (8, 5))
        sns.countplot(
            data = data,
            x = column_label,
            color = color
        )
        plt.title(f'Countplot of {title}')
        plt.ylabel('Count')
        plt.xticks(rotation = 90)
        plt.show()

    # Handle invalid graph type input
    else:
        print(
            "Please enter valid graph name: 'histogram_boxplot' or 'countplot'."
        )
        return

def anova_test(data, numerical_var, *categorical_vars):

    results = {}

    for cat_var in categorical_vars:
        """
        Perform an ANOVA test and create boxplots for each categorical variable.

        Parameters:
        - data (DataFrame): The DataFrame containing the data.
        - numerical_var (str): Name of the numerical variable.
        - *categorical_vars (str): Names of the categorical variables.
        """

        # Create groups for ANOVA test
        groups = [data[numerical_var][data[cat_var] == level].dropna()
        for level in data[cat_var].unique()]

        # Perform ANOVA
        f_stat, p_value = f_oneway(*groups)

        # Store results
        results[cat_var] = {
            'F-statistic': f_stat,
            'p-value': p_value
        }

        # Generate box plot
        colors = ['skyblue', 'salmon', 'orange', 'lightgreen', 'gold']
        random_color = random.choice(colors)

        plt.figure(figsize = (10, 6))
        sns.boxplot(
            x = numerical_var,
            y = cat_var,
            data = data,
            color = random_color
        )
        plt.title(f"Box Plot of {numerical_var} by {cat_var}")
        plt.xlabel(numerical_var)
        plt.ylabel(cat_var)
        plt.xticks(rotation = 0)
        plt.show()

        # Output results
        print(
            f"ANOVA test for {numerical_var} and {cat_var}:\n"
            f"F-Statistic: {f_stat:.4f}\n"
            f"P-Value: {p_value:.4f}\n"
        )

        if p_value < 0.05:
            print(
                f"There is a statistically significant difference "
                f"between {numerical_var} and {cat_var} (p < 0.05).\n"
            )
        else:
            print(
                f"There is no statistically significant difference "
                f"between {numerical_var} and {cat_var} (p < 0.05).\n"
            )
